# ShuntingYard

a simple math expression solver using the shunting yard algorithm

## Usage

```
|> x = 3 + 4 * 2 / (1 - 5)^2
|: 3.5
|> y = Pi * (x / E)
|: 4.045045724268226012685326141594437
|> 
```

use `?` or `help` to display additional commands and constants
